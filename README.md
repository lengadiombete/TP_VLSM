# TP Réseaux – VLSM et Plan d’Adressage IP

## Description
Ce travail pratique porte sur l’utilisation du VLSM (Variable Length Subnet Masking) afin de concevoir un plan d’adressage IP optimisé à partir d’un réseau de base.

## Objectifs du TP
- Comprendre le principe du VLSM
- Calculer des sous-réseaux à partir d’un /16
- Déterminer les adresses réseau, broadcast et plages d’hôtes
- Optimiser l’utilisation des adresses IP
- Vérifier la connectivité locale entre postes

## Technologies utilisées
- Réseaux TCP/IP
- VLSM
- Cisco Packet Tracer
- Postes clients et switches Cisco

## Travail réalisé
- Calcul de l’adresse réseau à partir de 10.10.23.67/21
- Conception d’un plan VLSM pour plusieurs départements
- Attribution des plages IP par département
- Analyse du gaspillage d’adresses
- Explication de l’absence de connectivité inter-réseaux sans routage

## Instructions
1. Calculer les sous-réseaux selon les besoins en hôtes
2. Attribuer les adresses IP aux postes
3. Tester la connectivité locale (ping)
4. Analyser les résultats
